### Simple Tactical RPG - Independent Study (Game Dev - 1 credit)

Retro-inspired tactical RPG built with React and CSS 3D transforms, featuring a dual-mode gameplay loop consisting of world exploration and turn-based grid combat. (Mostly Vibe Coded)

Highlights:

* Dual-State Gameplay: Seamless transition between a top-down "Exploration Mode" and a 3D isometic "Battle Mode" triggered by random encounters.
* Turn-Based Combat Engine: Fully functional initiative system with action economy (Move/Attack dots) and automated AI enemy behavior.
* Isometric 3D Rendering: Pure CSS transform-style: preserve-3d implementation to create a 3D depth effect for tiles, obstacles, and character sprites without external 3D libraries.
* Tactical Action System: Includes varied combat mechanics such as melee strikes, AOE "Splatter Shots" with area-of-effect previews, and path-range highlighting.

Logic:

* React Hooks (useState, useEffect) for managing complex game states and combat turns.

Engine:

* Custom-built grid logic supporting collision detection and Manahattan distance calculations for movement ranges.

Visuals:

* Advanced CSS transforms for isometric persepctive and 3D character billboards.
